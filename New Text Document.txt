Build a full Laravel CRUD final project called:

# GeoArchive — Georgian Historical Artifact and Event Management System

## Main Goal

Create a PHP Laravel web application that demonstrates all required course concepts:

* MVC Architecture
* Database migrations
* Eloquent ORM
* One-to-One relationship
* One-to-Many relationship
* Many-to-Many relationship
* Full CRUD operations
* Blade templating engine
* Authentication
* Middleware
* Form validation
* CSRF protection
* File uploads

Use only Laravel, PHP, Blade, HTML/CSS, and standard Laravel features. Do not build API-only. Do not use React/Vue. The project must be server-rendered with Blade.

---

# 1. Project Theme

The project is a Georgian history archive system.

Users can view Georgian historical artifacts, historical events, categories, and tags.

Admins can create, edit, update, and delete data.

Example artifacts:

* Georgian manuscript
* Colchis coin
* Svan tower
* King Tamar period item
* Medieval weapon
* Church icon
* Archaeological object

Example historical events:

* Battle of Didgori
* Golden Age of Georgia
* King Tamar period
* Russian annexation
* First Democratic Republic of Georgia

# 3. Database Tables and Relationships

Create migrations, models, controllers, routes, views, validation, and relationships.

## users table

Use Laravel default users table.

Add a `role` column:

```text
role enum/string default user
possible values: admin, user
```

---

## profiles table

This demonstrates One-to-One relationship.

Fields:

```text
id
user_id foreign key
bio nullable text
avatar nullable string
created_at
updated_at
```

Relationships:

```php
User hasOne Profile
Profile belongsTo User
```

Users can edit their own profile and upload avatar image.

---

## categories table

Fields:

```text
id
name string unique
description nullable text
created_at
updated_at
```

Relationship:

```php
Category hasMany Artifacts
Artifact belongsTo Category
```

---

## artifacts table

Fields:

```text
id
title string
description text
period string nullable
location string nullable
image nullable string
user_id foreign key
category_id foreign key
created_at
updated_at
```

Relationships:

```php
Artifact belongsTo User
Artifact belongsTo Category
Artifact belongsToMany Tag
```

This table demonstrates One-to-Many:

```php
User hasMany Artifacts
Category hasMany Artifacts
```

---

## tags table

Fields:

```text
id
name string unique
created_at
updated_at
```

---

## artifact_tag pivot table

This demonstrates Many-to-Many relationship.

Fields:

```text
id
artifact_id foreign key
tag_id foreign key
created_at
updated_at
```

Relationships:

```php
Artifact belongsToMany Tag
Tag belongsToMany Artifact
```

---

## historical_events table

Fields:

```text
id
title string
description text
date_or_period string nullable
location string nullable
image nullable string
created_at
updated_at
```

Admin can CRUD historical events.

Users can only view them.

# 8. Validation Rules

Use Laravel validation in controller methods.

Examples:

## Artifact validation

```php
title required max 255
description required
period nullable max 255
location nullable max 255
category_id required exists categories,id
image nullable image max 2048
tags nullable array
tags.* exists tags,id
```

## Category validation

```php
name required unique max 255
description nullable
```

## Tag validation

```php
name required unique max 255
```

## Event validation

```php
title required max 255
description required
date_or_period nullable max 255
location nullable max 255
image nullable image max 2048
```

## Profile validation

```php
bio nullable max 1000
avatar nullable image max 2048
```

Show validation errors in Blade.

# 9. File Upload Requirements

Store uploaded images in:

```text
storage/app/public/artifacts
storage/app/public/events
storage/app/public/avatars
```

Use:

```bash
php artisan storage:link
```

When updating an image, delete the old image if it exists.

When deleting an artifact/event/profile avatar, delete the related file if it exists.

Display images using:

```blade
<img src="{{ asset('storage/' . $artifact->image) }}">
```
